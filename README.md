Settings
------
Python 3.5.2 over<br>
Django 2.2.0 over 

Details
------
- Put your .pcd file in static/data/ directory.

- Have to run web browser private mode

- 
##Run

python3 manage.py runserver or ./manage.py runserver

##Address

https://\<IP:port_number\>/?load=/static/data/<file_name>.pcd
